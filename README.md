# Yazed — 3D Experiences

Interaktive 3D-Webseiten mit **Three.js** und **UnrealBloomPass**.

## Projekte

| Seite | Beschreibung |
|-------|-------------|
| [`3d-website.html`](./3d-website.html) | Torus-Knoten mit Glow, Ringen, Orbiting-Spheres und 4000 Partikeln |
| [`dark-mode-3d.html`](./dark-mode-3d.html) | Icosahedron mit Wireframe-Doppelschalen, 5000 Partikeln und orbitalen Lichtern |

## Features

- Vollständig clientseitig (kein Build-Tool nötig)
- Maus-Interaktion — 3D-Szene folgt der Mausbewegung
- Bloom-Glow-Effekte via UnrealBloomPass
- Partikelsysteme mit sanfter Drift-Animation
- Responsive & Fullscreen
